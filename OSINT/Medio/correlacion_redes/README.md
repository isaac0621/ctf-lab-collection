# OSINT - Correlación de Redes (OSINT - Medio)

## Descripción
Se te proporciona varios usernames e identidades. Debes correlacionar la información para conectar a las personas.

## Instrucciones
1. Recopila información de múltiples plataformas
2. Busca conexiones (amigos, seguidores mutuos, etc.)
3. Crea un mapa de relaciones
4. Identifica patrones

## Técnicas
- Seguir relaciones sociales
- Analizar metadatos
- Correlacionar timestamps
- Identificar patrones de comportamiento
- Geolocalización

## Herramientas
- `Maltego` - Análisis de información
- `OSINT Framework`
- `Kali Linux` con herramientas OSINT
- Google Dorks avanzado

```
CTF{network_correlation}
```

## Conceptos

- Relationship mapping
