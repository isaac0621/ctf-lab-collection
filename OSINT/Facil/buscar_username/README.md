# OSINT - Búsqueda de Username (OSINT - Fácil)

## Descripción
Se te proporciona un username. Tu misión es encontrar información sobre esa persona en internet.

## Instrucciones
1. Búscale en redes sociales comunes
2. Intenta encontrar una presencia en línea
3. Recopila información pública
4. Encuentra la flag

## Sitios útiles
- GitHub
- Twitter
- LinkedIn
- Reddit
- Instagram
- TikTok
- Twitch
- YouTube

## Herramientas
- `sherlock` - Búsqueda de usernames en múltiples plataformas
- Google Dorks
- Whoami API

## Ejemplo
```bash
pip install sherlock-project
sherlock username
```

## Flag
```
CTF{osint_found}
```
