# CONTRIBUYE.md - Guía de Contribución

## ¿Cómo contribuir?

### 1. Reportar un bug
- Abre un [issue](https://github.com/isaac0621/ctf-lab-collection.git)
- Sé descriptivo en el título
- Incluye detalles del error

### 2. Agregar una solución
```bash
git checkout -b solucion/categoria/nombre
# Añade tu solución
git commit -m "Add solution for CATEGORIA/Nivel/nombre"
git push origin solucion/categoria/nombre
```

### 3. Mejorar documentación
- Correcciones de typos
- Ejemplos más claros
- Mejores explicaciones

### 4. Crear nuevo reto
- Sigue el template de `PROTOCOLO.md`
- Crea una rama `feature/nuevo-reto`
- Incluye README, checker, solution

## 📋 Checklist para PR

- [ ] Código funciona
- [ ] README está actualizado
- [ ] Checker verifica correctamente
- [ ] Sin archivos secretos
- [ ] Sigue el protocolo
