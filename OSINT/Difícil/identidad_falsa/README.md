# OSINT - Identidad Falsa Completa (OSINT - Difícil)

## Descripción
Se te proporciona datos de una "persona" que podría ser una identidad falsa.
Tu misión es:
1. Investigar toda su presencia digital
2. Encontrar inconsistencias
3. Probar que es falsa
4. Recopilar evidencia

## Técnicas avanzadas
1. **Análisis de fotos**
   - Reverse image search
   - EXIF data
   - Deepfake detection

2. **Análisis de comportamiento**
   - Patrones de posting
   - Zonas horarias
   - Idiomas
   - Referencias culturales

3. **Verificación de documentos**
   - Búsqueda de registros públicos
   - Verificación de teléfono
   - Cross-reference de datos

4. **Análisis técnico**
   - IP addresses
   - Dispositivos
   - Navegadores
   - Cookies

## Herramientas
- Google Dorks
- Shodan
- Reverse image search
- WayBack Machine
- WHOIS
- DNS lookup
- Email verification

## Red flags
- Foto de perfil genérica
- Biografía inconsistente
- Demasiada perfección
- Errores ortográficos recurrentes
- Cuentas nuevas con mucha actividad
- Seguimiento selectivo
